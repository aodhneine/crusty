# crusty
Experimental Rust compiler written in pure C. It is fully compliant with latest
Rust specification.

_Currently does not support Windows, but support for it may be added in the
near future._


## Build

```console
$ make
$ ./crusty examples/main.rs
$ ./main
```

## Dependencies
C99 compiler, ld and make.
